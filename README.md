# BitCoin AI Trading

비트코인 자동매매 프로그램입니다. 다양한 기술적 지표와 시장 데이터를 분석하여 업비트에서 비트코인을 자동으로 매매합니다.

## 주요 기능

- 다양한 기술적 지표 분석 (이동평균선, RSI, MACD, 볼린저 밴드 등)
- 시장 데이터 분석 (호가창, 체결 데이터, 김프 등)
- 시장 심리 분석 (공포&탐욕 지수 등)
- Claude AI 통합 (선택 사항)
- 커스터마이징 가능한 매매 전략
- 상세한 거래 로깅 및 분석
